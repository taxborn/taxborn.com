# :v: [taxborn.com](https://www.taxborn.com) - Portfolio
My personal website / portfolio in the style of a [UNIX man page](https://en.wikipedia.org/wiki/Man_page). I felt like this
was appropriate as I love the command line, love parsing through `man` pages, and gives me some freedom from constructing
a pixel-perfect website (which I am not great at, front-end is kinda tough, lots of respect to front-end developers).

## Running
This website is built with the [Astro](https://astro.build) framework, as that was really easy for me to pick up and there
wasn't a whole lot of learning requried for me to get up and running.
```
npm i && npm run dev
```
should be all you need to run this website locally.
